# Repositorio de la Tarea 1- Ramirez Aguirre Jose Alfredo
UIFragmentsApp

Descripción de la aplicación

Esta aplicación Android fue desarrollada para demostrar el uso de Activities y Fragments en la construcción de interfaces de usuario dentro de Android Studio.

La aplicación cuenta con una Activity principal que contiene un menú de navegación inferior, el cual permite cambiar entre diferentes Fragments. Cada Fragment muestra distintos elementos de interfaz de usuario utilizados comúnmente en el desarrollo de aplicaciones móviles.

Fragments implementados

Fragment 1 – TextFields

Este fragment demuestra el uso de campos de texto (EditText). Los EditText permiten capturar información del usuario, como nombre o correo electrónico. En este fragment el usuario puede escribir datos y mostrarlos mediante un botón.


Fragment 2 – Botones

Este fragment muestra diferentes tipos de botones utilizados en Android:

• Button
• ImageButton
• FloatingActionButton

Cada botón ejecuta una acción al presionarse y muestra un mensaje Toast indicando que fue presionado.


Fragment 3 – Elementos de selección

Este fragment demuestra el uso de elementos de selección:

• CheckBox
• RadioButton
• Switch

El usuario puede seleccionar opciones y presionar un botón para ver un resumen de las selecciones realizadas.


Fragment 4 – Listas

Este fragment utiliza RecyclerView para mostrar una lista de elementos. RecyclerView permite mostrar grandes cantidades de información de manera eficiente y organizada.

Cada elemento de la lista puede ser seleccionado y muestra un mensaje Toast indicando el elemento seleccionado.


Fragment 5 – Elementos de información

Este fragment muestra elementos utilizados para presentar información al usuario:

• TextView
• ImageView
• ProgressBar

El usuario puede presionar un botón para aumentar el progreso de la barra y visualizar cómo cambia su estado.

Instrucciones para ejecutar la aplicación

1. Clonar el repositorio desde GitHub.
2. Abrir el proyecto en Android Studio.
3. Esperar a que el proyecto sincronice las dependencias de Gradle.
4. Ejecutar la aplicación en un emulador o dispositivo Android.
5. Utilizar el menú inferior para navegar entre los diferentes Fragments.
